## 快速开始

### 克隆项目

![1](readmeme/clone.png)

You know M3?

### 安装依赖

```bash
npm install
```

### 启动服务

```bash
npm start
```

服务将运行在 `http://localhost:3000` ，然后就可以用了。
![zimin](readmeme/zimin.png)
模型名称和 API Key 埃及吧写啥写啥，都能给你中转过去。

### 配置

![mortis](readmeme/mortis.png)

如果你真的要改些什么的话，

你可以复制 `.env.example` 为 `.env` 并修改端口：

```bash
cp .env.example .env
```
